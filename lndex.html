Introduction

This web page is created as part of my TVET CDACC Level 6 web development course. It demonstrates how to use HTML5 semantic tags to improve structure, accessibility, and SEO.

Accessibility

Using semantic elements like <header>, <nav>, <main>, and <footer> helps screen readers navigate the page easily. Landmarks and ARIA labels enhance user experience for those using assistive technologies.

SEO Basics

Good SEO starts with proper use of heading tags, descriptive content, and meaningful structure. This page uses metadata, organized sections, and accessible links to improve visibility to search engines.

Conclusion

By combining HTML5 structure, accessibility principles, and basic SEO techniques, this page lays the foundation for a user- and search-friendly web experience.
